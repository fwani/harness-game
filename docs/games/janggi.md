# 장기 (Janggi) 사양

> 규칙 + 구현 상태 + UI/UX 요구사항. 상위 인덱스: [README.md](README.md).
> 도메인·애플리케이션(이동·장군·외통)에 더해 **UI 플레이까지 연동 완료**
> (기물 선택·합법 수 표시·이동·턴 전환·장군 경고·승부·새 게임).

## 1. 개요

- **장르**: 2인 로컬(핫시트) 보드 게임.
- **보드**: 9×10 (가로 `WIDTH=9`, 세로 `HEIGHT=10`). 좌표 `board[y][x]`, y:0..9, x:0..8.
- **진영**: `han`(한 漢, 위쪽 y 작은 쪽) · `cho`(초 楚, 아래쪽 y 큰 쪽).

## 2. 규칙

### 좌표·배치
- `han` 1선 y=0, 궁성 y=0..2, 장 (4,1), 포 y=2, 병 y=3.
- `cho` 1선 y=9, 궁성 y=7..9, 장 (4,8), 포 y=7, 졸 y=6.
- 진영당 16개(장1·사2·상2·마2·차2·포2·졸5), 합 32개. 표준 차림 1종만(상마 교체 변형 제외).

### 기물 이동 (`isLegalMove`)
- **장·사**: 궁성 안에서 직선 1칸, 궁성 대각 라인에서 대각 1칸. 궁성 밖 금지.
- **차**: 직선 임의 칸(경로 막힘 없음) + 궁성 대각 라인 슬라이드.
- **포**: 직선으로 받침(다리) **정확히 1개**를 넘어 이동/포획. 포는 받침·대상이 될 수 없다.
- **마**: 직진 1 + 대각 1, 직진 지점(멱)이 막히면 불가.
- **상**: 직진 1 + 대각 2, 경로(직진 1 + 대각 첫 칸)가 막히면 불가.
- **졸·병**: 전진/좌우 1칸(후퇴 불가), 궁성 안에서 전진 대각 1칸. (han은 +y, cho는 -y 전진)
- 공통: 보드 밖·제자리·아군 위 이동 금지. 상대 기물 위는 포획.

### 장군·외통 (`isInCheck` / `isCheckmate`)
- **장군**: 상대 어떤 기물이든 내 장(general) 위치로 가는 합법 수가 하나라도 있으면 장군.
  내 장이 보드에 없으면(이미 잡힘) 장군으로 간주.
- **외통**: 장군이고, 어떤 합법 수를 둬도 장군을 벗어나지 못하면 외통수.
- **범위 밖(도메인 주석 명시)**: 빅장(bikjang 마주보기 무승부), 차림 변형, 턴 오케스트레이션.
