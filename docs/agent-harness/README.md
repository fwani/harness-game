## What this repo is

플래시 게임 레파지토리임. 카드게임, 홀짝게임, 오목, 바둑, 장기 등 간단하게 게임을 할 수 있고, 멀티로 진행도 되며, 기록을 할 수 있는 게임시스템임.

> Status가 비어 있으면 `docs/agent-harness/PRODUCT_CONTEXT.md`에서 목적/사용자를 채운다.

## Quick commands

- Install: `npm ci`
- Run (web UI): `npm run dev` (Vite, 기본 http://localhost:5173)
- Run (CLI): `npm start`
- Test: `npm test`
- Full verification: `scripts/agent-harness/agent-verify`
