# Active execution plans

> Generated by company-agent-harness v0.4.0.

진행 중인 작업의 실행 계획을 한 파일씩 둔다. 복잡하거나 위험한 변경은 코드 수정 전에 여기에 계획을 먼저 남긴다.

## Plan file format

```
# <작업명>

## Goal (한 문장)
## Files to change
## Steps
## Verification
## Risks / rollback
```

완료되면 `../completed/`로 옮긴다.
