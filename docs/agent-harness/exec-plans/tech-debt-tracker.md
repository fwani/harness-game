# Tech Debt Tracker

> Generated by company-agent-harness v0.4.0.

알려진 기술 부채를 한 곳에 모아 추적한다. 이 파일이 유지되면 하네스 성숙도 Level 5 신호가 된다.

| ID | 항목 | 영향 | 제안 조치 | 상태 |
| --- | --- | --- | --- | --- |
| TD-001 | (예) 테스트 커버리지 공백 | 중 | 회귀 테스트 추가 | open |

## 사용 규칙

- 새 부채를 발견하면 행을 추가한다.
- 큰 항목은 `active/`에 실행 계획을 만든다.
- 해결되면 상태를 `done`으로 바꾸고 근거(PR/커밋)를 남긴다.
