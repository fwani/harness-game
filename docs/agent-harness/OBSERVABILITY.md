## How an agent reproduces a failure

1. 실패한 명령/요청을 한 문장으로 정리한다.
2. 로컬에서 재현 시도: `scripts/agent-harness/agent-debug`.
3. 로그를 확인하고 관련 라인을 인용한다.
4. 최소 재현 케이스를 테스트로 추가한다.
