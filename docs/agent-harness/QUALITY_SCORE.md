# Quality Score / Self-review

> Generated by company-agent-harness v0.4.0.
> 변경 후 `scripts/agent-harness/agent-review`를 실행하면 변경 요약과 함께 이 체크리스트를 출력한다.

## Review checklist

### Scope
- [ ] 요청 범위를 벗어난 변경이 없는가?

### Correctness
- [ ] 핵심 동작을 검증했는가?

### Tests
- [ ] 코드 변경에 대응하는 테스트가 있는가?

### Docs
- [ ] 동작/API/설정이 바뀌었다면 문서를 업데이트했는가?

### Security
- [ ] 비밀정보가 포함되지 않았는가?
- [ ] 인증/인가 우회가 없는가?

### Operations
- [ ] 배포, 마이그레이션, 롤백 영향이 있는가?
