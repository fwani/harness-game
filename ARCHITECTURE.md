## Forbidden imports

> Status: 확정. `scripts/agent-harness/agent-arch-check`(= `agent-harness verify`)가 검사한다.

- `src/domain` 은 `application` 을 import 하지 않는다.
- `src/domain` 은 `infrastructure` 를 import 하지 않는다.
- `src/application` 은 `infrastructure` 를 import 하지 않는다.

```yaml
architecture:
  forbidden_imports:
    - from: "src/domain"
      pattern: "infrastructure"
      reason: "domain must not depend on infrastructure"
```
